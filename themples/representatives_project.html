Political Representatives APP

A Political Preparedness app that will deliver civic data to end users via the app. Using the Google Civic API you can see your representatives and the upcoming elections.

Code Source
  • Languages Kotlin, XML
  • Arquitecture MVVM
  • Tecnologies JQuery, Room, LiveData, Navigation, Retrofit, OKhttp, Google Civic API, Glide, Location Services, RecyclerView, DataBinding, MotionLayout
  • Softwares Android Studio, Postman
